# 🌆 MH2 City X

**MH2 City X** is a browser-based 3D open-world city simulation project built with Three.js. It combines FPS-style movement, 3D graphics, and math-based physics to create an interactive virtual world that can be expanded into a full city system.

---

## 🧠 How It Works (Project Logic)

This project is built on core **3D math + game engine concepts**:

- 🎯 **Camera System (Player View)**  
  The camera acts like the player’s eyes in a 3D space.

- 📐 **Trigonometry (sin, cos)**  
  Used to calculate direction for movement and rotation.

- 🧭 **Vector Math (x, y, z)**  
  Controls movement in 3D space (forward, backward, left, right).

- 🔄 **Game Loop (Animation Frame)**  
  The world updates continuously using `requestAnimationFrame()`.

- 🖱️ **Mouse + Keyboard Controls**  
  Mouse controls camera rotation, keyboard controls movement.

👉 In short:  
**Math + Physics + Vectors = Real-time 3D World**

## 🎮 Features (Current)
- FPS-style movement (WASD + Arrow keys)
- Mouse look system (yaw & pitch rotation)
- Starfield 3D sky environment
- Ground plane (world base)
- 3D text rendering using Three.js
- Smooth real-time rendering engine
