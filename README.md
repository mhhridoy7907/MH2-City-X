# 🌆 MH2 City X

**MH2 City X** is a browser-based 3D open-world city simulation project built with Three.js. It features FPS-style movement, immersive 3D environments, and is designed for future expansion including cars, buildings, and a full interactive city system.

---

## 🚀 Features
- 🎮 FPS-style movement (W, A, S, D + Mouse look)
- 🌌 Real-time 3D environment
- ⭐ Starfield sky effect
- 🔤 3D text rendering (Three.js)
- 🏙️ Expandable open-world structure

---

## 🛣️ Future Updates
- 🚗 Drivable 3D cars
- 🏢 Procedural buildings & city system
- 🚦 Roads & traffic AI
- 🌍 Full open-world map
- 🌦️ Day/Night & weather system
- 🤖 AI integration (MH2 AI)

---

## 🧠 Tech Stack
- Three.js (WebGL)
- JavaScript (ES6)
- HTML5 & CSS3

---

## 🎮 Controls
- **W A S D** → Move
- **Mouse Drag** → Look around
- **Arrow Keys** → Additional movement/rotation
